# UI Components

Shared [shadcn-svelte](https://www.shadcn-svelte.com/) components.

## Setup

add/update `shadcn` components

Run this command and select `shadcn` components you want to add/update

```shell
cd packages/ui
pnpm run ui:add --all
```
